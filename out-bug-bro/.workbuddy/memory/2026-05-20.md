## 第三批修复

- 根因: _on_game_over() 调用 SaveManager.reset_run() → clear_runtime() → CurrencyManager.currency = 0，复活时已无金币
- 修复: 移除 _on_game_over 中的 reset_run()，改到 _on_restart_pressed / _on_menu_pressed 中调用
- 复活按钮现在能看到实际金币
