## 修复：ArenaBounds class_name 与 Autoload 冲突

- **修复**: 删除 `class_name ArenaBounds`，改 `extends Node2D` → `extends Node`

- 玩家位置 `(-200,0)` → `(0,0)`，Boss 位置改为 `(300,0)`
