   ```
   [resource]
   script = ExtResource("1_")
   enemy_hp_level_mult = 1.08
   ...
   ```
   - 去掉 `script_class`（不存在的属性）
   - 去掉 `[sub_resource type="WaveConfig"]`（错误写法）

   - `%TextLabel` → `$CenterContainer/Panel/VBox/TextLabel`
   - `%IconLabel` → `$CenterContainer/Panel/VBox/IconLabel`

- Godot 4 的 `StyleBoxFlat` 没有 `corner_radius_all` 属性，只有 `set_corner_radius_all()` 方法
- 手写 .tscn 时容易漏掉 `unique_name_in_owner = true`，导致 `%` 引用找不到节点
- `_ready()` 中向父节点 `add_child()` 可能触发 "busy setting up children" 错误，需 `call_deferred`

## 2026-05-27 (续4) — .tscn GDScript 语法混入修复

### 根因
手写 `.tscn` 文件中混入了 GDScript 方法调用语法，Godot 解析器不识别，导致节点创建失败：
- `add_theme_constant_override("separation", 16)` → 应为 `theme_override_constants/separation = 16`
- `add_theme_font_size_override("font_size", 28)` → 应为 `theme_override_font_sizes/font_size = 28`
- `add_theme_color_override("font_color", Color(...))` → 应为 `theme_override_colors/font_color = Color(...)`

### 修复文件
1. **`scenes/ui/tutorial_overlay.tscn`** — 5 处 GDScript 方法调用改为 .tscn 属性语法
2. **`scenes/ui/meta_upgrades.tscn`** — 同上 4 处，同时修复非法 UID `uid://meta_upgrades_gd`

### .tscn 主题覆写正确语法速查
| GDScript 方法 | .tscn 属性 |
|---|---|
| `add_theme_constant_override("separation", 16)` | `theme_override_constants/separation = 16` |
| `add_theme_font_size_override("font_size", 28)` | `theme_override_font_sizes/font_size = 28` |
| `add_theme_color_override("font_color", Color(...))` | `theme_override_colors/font_color = Color(...)` |
| `add_theme_stylebox_override("panel", ...)` | 不推荐在 .tscn 中用，改在 GDScript 中设置 |
